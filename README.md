# Mathematical-Modeling-Project
Predict and forecast the population growth of Singapore based on the past population trends between 1964 and 2024. There are three models of population growth that are discussed, which include the Malthusian (exponential) growth model, the Logistic growth model, and the Geometric growth model.

📌 Project Overview
This project involves the construction and analysis of mathematical models to represent real-world systems. By translating physical or business scenarios into mathematical language, this study aims to predict future outcomes and optimize decision-making processes.

🛠 Modeling Framework
The project follows a structured modeling workflow:
- Problem Formulation: Defining variables and establishing the relationship between dependent and independent factors.
- Mathematical Techniques: Used to simulate the system.
- Assumptions & Constraints: Identifying the boundaries and conditions under which the model remains valid.
- Validation: Comparing model predictions against historical or observed data to ensure reliability.

📈 Key Insights
- Predictive Power: Developed a model capable of forecasting trends based on specific input parameters.
- Sensitivity Analysis: Evaluated how changes in key variables impact the overall system output.
- Strategic Application: Provided data-driven recommendations based on the mathematical results.

🎓 Academic Context
This research was conducted as part of the Business Mathematics degree at Universiti Utara Malaysia (UUM). It highlights the application of quantitative methods to solve complex, multi-variable problems.
