# Mathematical-Modeling-Project
Predict and forecast the population growth of Singapore based on the past population trends between 1964 and 2024. There are three models of population growth that are discussed, which include the Malthusian (exponential) growth model, the Logistic growth model, and the Geometric growth model.

📌 Project Overview
This project focuses on the application of numerical methods to solve complex mathematical problems that are otherwise difficult or impossible to solve analytically. It demonstrates the implementation of iterative algorithms to find roots of transcendental equations and solutions to systems of linear equations.

🛠 Methods ImplementedThe project explores the following numerical techniques:
- Root-Finding Algorithms: used to find the zeros of non-linear functions.
- Linear Systems: (e.g., Gauss-Seidel, Jacobi, or LU Decomposition) used to solve multi-variable equations.
- Error Analysis: Evaluation of absolute and relative errors to ensure the convergence and accuracy of the results.

📊 Key Highlights
- Efficiency: Analyzed the rate of convergence for different iterative methods.
- Accuracy: Achieved a high degree of precision for final solutions.
- Application: Applied these mathematical models to solve practical engineering or business-related scenarios.

🎓 Academic Context
This project was completed as part of the Business Mathematics curriculum at Universiti Utara Malaysia (UUM). It showcases the bridge between theoretical mathematics and computational implementation.
